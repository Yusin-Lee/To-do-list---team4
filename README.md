# To-Do List Manager

Shell 기반 할일 관리 프로그램

## 팀원
- 202501386 이유신
- 202501370 오현준

## 기능
- Feature 1: 할일 추가/삭제 기능
- Feature 2: 할일 목록 조회 및 완료 표시
- Feature 3: 파일 저장/불러오기
- Feature 4: 검색 및 필터링 기능

## 사용법
```bash
./todo.sh
```

## 명령어
- `add [내용]` - 새로운 할일 추가
- `delete [ID]` - 할일 삭제
- `list` - 전체 할일 목록 보기
- `complete [ID]` - 할일 완료 처리
- `help` - 도움말 보기
- `exit` - 프로그램 종료
